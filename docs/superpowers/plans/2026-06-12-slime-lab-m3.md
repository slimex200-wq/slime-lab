# SLIME LAB M3 — 깊이 구현 계획

> M1/M2 컨벤션 유지 (TDD·시각 검증·`git add -A`·astro check·데몬 스크린샷). 스펙 §11 M3 = THE DIG, 터미널 이스터에그, 사운드, OG 동적 이미지, 모바일/접근성 마감 + 알려진 항목(LCP, Worker 핸들러 테스트).

**Goal:** 원페이지에 시간의 깊이(THE DIG)와 숨은 레이어(터미널)를 더하고, 사운드·동적 OG·성능을 제출 가능 수준으로 마감한다.

### Task 1: THE DIG — 지층 시추 섹션
**Files:** Modify `src/pages/index.astro`, `src/styles/global.css`
- 인덱스 섹션 아래에 `s4-wrap`(높이 ~320vh) + sticky 100vh 뷰포트. 레이어 4장: TOPSOIL(live, 데이터에서 이름 나열) → SEDIMENT(beta) → BEDROCK(2025~) → FOSSIL RECORD(archived/frozen † 카드, `/specimen/<slug>/` 링크). 콘셉트 프로토타입(concepts-all.html §4)의 변환식 이식: 스크롤 진행도 p → `translateY(-p×(L-1)×100vh)` + 우측 깊이 HUD(`(p×48).toFixed(1)m`) + 눈금자. 자연 스크롤 — hijack 금지.
- reduced-motion: sticky 해제, 레이어를 일반 세로 흐름으로.
- 검증: 데몬 스크롤 3지점 스크린샷 + 콘솔 0.

### Task 2: 터미널 이스터에그
**Files:** Create `src/engine/terminal.ts`, Modify `src/pages/index.astro`, `global.css`
- ` (백틱) → 고정 오버레이 터미널, ESC 닫기. 명령: `help`, `ls /specimens`(컬렉션), `status <slug>`(frontmatter+라이브 vitals), `autopsy <slug>`(archived만), `uptime`, `clear`. 데이터는 빌드 시 `<script type="application/json" id="lab-data">`로 주입.
- `initTerminal(data)` 모듈 — DOM 생성/입력 파싱/히스토리(↑). 순수 파서 `parseCmd(input, data): string[]`는 vitest 4케이스(미지 명령/status 정상/autopsy 비대상 거부/ls 개수).
- 발견 힌트: 히어로 hint 줄에 `` PRESS ` ``.

### Task 3: 사운드
**Files:** Create `src/engine/sound.ts`, Modify `mount.ts`, `index.astro`
- v8 thump(사인 52→34Hz) 이식. 히어로 bgroup에 `SOUND ○ OFF` 자석 버튼(기본 OFF, AudioContext는 첫 토글에서 생성). mount의 `onBeat` 훅에 연결, 쇼크는 gain 1.0. 스크롤 페이드<0.5면 박동음 끔.

- **타임박스 60분**: wasm 번들/패키지 문제로 막히면 폴백 — 빌드 시 데몬 스크린샷으로 `public/og.png` 생성(배포 시점 BPM 박힘) + 메타 태그. 어느 쪽이든 og:title/description/twitter 카드 메타 추가.

### Task 5: LCP 개선 (디자인 보존)
