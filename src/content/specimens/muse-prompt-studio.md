### What it is

Muse Prompt Studio is a web-based prompt workbench for generative-AI power users, running in production at muse-prompt.app.

### The experiment

It started as a pain point inside my own generation workflow: iterating on prompts by hand didn't scale, so the workbench grew into its own product. The frontend ships on Cloudflare Pages with an API on Google Cloud Run, and billing runs through Paddle.

### Findings

Launched and live, with the Paddle billing cutover complete.
